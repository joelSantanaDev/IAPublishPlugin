# Contribuindo para o IA Publish Plugin

Obrigado por considerar contribuir para o IA Publish Plugin! 🎉

## 🐛 Reportando Bugs

Se você encontrou um bug, por favor abra uma issue incluindo:

- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs. atual
- Versão do WordPress e PHP
- Logs relevantes (se disponível)

## 💡 Sugerindo Funcionalidades

Sugestões são bem-vindas! Abra uma issue com:

- Descrição detalhada da funcionalidade
- Casos de uso
- Exemplos de como funcionaria

## 🔧 Pull Requests

1. Fork o repositório
2. Crie uma branch para sua feature (`git checkout -b feature/MinhaFeature`)
3. Commit suas mudanças (`git commit -m 'Adiciona MinhaFeature'`)
4. Push para a branch (`git push origin feature/MinhaFeature`)
5. Abra um Pull Request

### Diretrizes de Código

- Siga os padrões de código WordPress
- Comente código complexo
- Teste suas mudanças
- Atualize a documentação se necessário

## 📝 Padrões de Commit

Use mensagens de commit claras:

- `feat: adiciona nova funcionalidade`
- `fix: corrige bug X`
- `docs: atualiza documentação`
- `refactor: refatora código Y`
- `test: adiciona testes`

## 🧪 Testando

Antes de submeter um PR:

1. Teste em ambiente local
2. Verifique compatibilidade com WordPress 5.0+
3. Teste com diferentes provedores de IA
4. Verifique logs de erro

## 📄 Licença

## ❓ Dúvidas?

Abra uma issue com a tag `question` ou entre em contato.

Obrigado! 🙏
